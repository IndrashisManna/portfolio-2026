Assignment Questions & Answers

As part of my HTML/CSS assignment, I have answered all the required questions below. This page explains the design choices, implementation decisions, and technical concepts used in building this portfolio website.

1. What are the two main sections of an HTML file?

The two main sections of an HTML file are:

  • <head> section: Contains metadata like title, character encoding, linked CSS files, JavaScript files, and fonts.
  • <body> section: Contains visible content such as text, images, links, forms, and layout elements.

In this website, the <head> section includes meta tags, CSS links, and Google Fonts, while the <body> section contains all visible elements.

2. Did you separate your HTML and CSS into separate files?

Yes, HTML and CSS are separated into different files.

  • Improves readability and maintainability
  • Allows reuse of styles across pages
  • Improves performance through caching
  • Makes debugging easier

The CSS is stored in style.css and linked to all HTML pages.

3. Which HTML tag did you use to link HTML and CSS?

I used the <link> tag inside the <head> section.

<link rel="stylesheet" href="style.css">

This ensures styles are loaded before the page content is rendered.

4. If not separated, which tags could specify CSS?

Not applicable, as CSS is separated.

  • <style> tag for internal CSS
  • style attribute for inline CSS
5. In which section did you put the CSS link?

The CSS link is placed inside the <head> section.

  • <meta> tags
  • <title>
  • <link> and <script> tags
